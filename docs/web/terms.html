利用規約

最終更新日: 2026年4月1日

第1条（適用）

本利用規約（以下「本規約」）は、TrustCar（以下「当サービス」）の利用条件を定めるものです。ユーザーは本規約に同意のうえ、当サービスをご利用ください。

第2条（利用登録）

  1. 利用希望者は本規約に同意し、当サービス所定の方法により利用登録を申請します。
  2. 当サービスは、利用登録申請者に以下の事由があると判断した場合、登録を拒否できます。
    • 虚偽の情報による申請
    • 過去に本規約違反により登録を取り消されたことがある場合
    • その他、当サービスが不適切と判断した場合

第3条（アカウント管理）

  1. ユーザーはログイン情報（メールアドレス・パスワード）を自己責任で管理してください。
  2. アカウントの第三者への譲渡・貸与・共有は禁止します。
  3. 不正利用を発見した場合は速やかに当サービスにご連絡ください。

第4条（禁止事項）

ユーザーは以下の行為をしてはなりません。

  1. 法令または公序良俗に違反する行為
  2. 当サービス・他のユーザー・第三者の権利・利益を侵害する行為
  3. 虚偽の情報を登録・投稿する行為
  4. スパム・迷惑行為・嫌がらせ行為
  5. 当サービスのシステムへの不正アクセス・改ざん行為
  6. 商業目的の無断広告・宣伝行為
  7. その他、当サービスが不適切と判断する行為

第5条（投稿コンテンツ）

  1. ユーザーが投稿したコンテンツの著作権はユーザーに帰属します。
  2. ユーザーは当サービスに対し、コンテンツを無償で利用する権利を許諾します。
  3. 以下のコンテンツの投稿は禁止します。
    • 第三者の著作権・肖像権・プライバシーを侵害するもの
    • 差別的・暴力的・性的なもの
    • 虚偽・誇大な内容を含むもの
    • スパム・広告目的のもの
  4. 当サービスは、規約違反のコンテンツを予告なく削除できます。

第6条（位置情報の利用）

  1. ドライブログ機能の利用にはGPS位置情報の取得許可が必要です。
  2. 収集した位置情報はドライブログの記録・表示にのみ使用します。
  3. 位置情報の利用はデバイス設定から停止できます。
  4. 位置情報を第三者に販売・提供することはありません。

第7条（サービスの変更・中断・終了）

  1. 当サービスは、ユーザーへの事前通知なしにサービス内容を変更・追加・削除できます。
  2. システムメンテナンス・障害等によりサービスを一時中断することがあります。
  3. サービス終了の際は、30日前にアプリ内またはメールで通知します。

第8条（免責事項）

  1. 当サービスは、ユーザーの利用により生じた損害について、一切の責任を負いません。
  2. 当サービスは、投稿情報の正確性・安全性を保証しません。
  3. ユーザー間のトラブルについては当事者間で解決するものとし、当サービスは関与しません。
  4. システム障害・データ消失による損害について、当サービスは責任を負いません。

第9条（退会）

  1. ユーザーはアプリ内の退会機能からいつでも退会できます。
  2. 退会後30日間はデータが保持され、その後削除されます。
  3. 退会後のデータ復元はできません。

第10条（規約の変更）

当サービスは、必要に応じて本規約を変更できます。変更後の規約はアプリ内またはこのページに掲載した時点で効力を生じます。重要な変更については事前にお知らせします。

第11条（準拠法・管轄裁判所）

本規約の解釈には日本法を適用し、当サービスに関する紛争については東京地方裁判所を第一審の専属的合意管轄裁判所とします。
