## 詳細説明（4000文字以内）

```
TrustCar（トラストカー）は、愛車との時間をもっと豊かにするカーライフ管理アプリです。

■ 愛車管理
複数台の車両を登録して一元管理。メーカー・車種・年式・走行距離・車検日などを記録し、車検期限が近づくと通知でお知らせします。車検証をカメラで撮影するだけで情報を自動入力するOCR機能搭載。

■ 整備記録
オイル交換・タイヤ交換・法定点検など、すべての整備履歴を記録。請求書をスキャンして自動入力、PDF出力にも対応。いつ・どこで・いくらかかったかを見える化して、維持費の管理に役立てましょう。

■ ドライブログ
GPSで走行ルートを自動記録。走行距離・最高速度・平均速度を自動集計します。ドライブスポットの登録・評価機能で、お気に入りのルートを共有できます。

■ カーライフSNS
同じ車好きとつながるSNS機能。愛車の写真・カスタム・整備記録を投稿して、いいね・コメントで交流しましょう。ハッシュタグで同じ車種・メーカーのオーナーを探せます。フォロー機能で気になるユーザーの投稿をフォローできます。

■ マーケットプレイス
信頼できる整備工場・カーショップを検索。写真・サービスメニュー・価格を確認して、アプリから直接問い合わせできます。パーツのおすすめ機能で、愛車に合った部品を見つけましょう。

■ 安全・安心
データはGoogle Cloud（Firebase）で安全に管理。メールアドレスとパスワードで登録するだけですぐ使えます。

【対応車種】
国産車・輸入車を問わず、すべての自動車に対応しています。

【動作環境】
iOS 13.0以降 / Android 8.0以降
```
