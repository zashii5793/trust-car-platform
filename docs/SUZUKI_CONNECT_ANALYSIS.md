# スズキコネクト vs Trust Car Platform — 競合分析と強み強化戦略

作成日: 2026-06-27
対象: スズキコネクト（SUZUKI CONNECT）との機能比較と差別化方針

---

## 0. 結論（先に要点）

**両者は同じ「車のアプリ」だが、競合軸がそもそも違う。**

- **スズキコネクト** = テレマティクス。車載通信機（ハードウェア）を前提に、「乗っている今この瞬間」のリアルタイム体験を提供する。**スズキ新車・対応車種に縛られる。**
- **Trust Car Platform** = データ／ソフトウェア。スマホ単体で、**全メーカー・全年式・中古車を含む「所有のライフサイクル全体」**（購入前 → 整備 → 売却）を支える。

→ **テレマティクスで正面から戦うべきではない（ハード非保有で勝てない）。**
むしろスズキコネクトが構造的に持てない領域 ——「全車種横断」「整備履歴の資産化」「コミュニティ」「B2Bマッチング」「法人フリート」—— を一気に深掘りするのが勝ち筋。

---

## 1. スズキコネクトの機能像

月額550円（税込）／初度検査から36か月無料。車載通信機を積んだ対応車種が前提。

### 1-1. オペレーターサービス（安心・ハードウェア依存）
- **自動緊急通報** — エアバッグ展開を検知し自動でオペレーターへ。緊急車両出動を要請。
- **SOSボタン（手動通報）** — 24時間365日。消防・警察・保険会社・JAFへ取次。

### 1-2. アプリサービス（快適・便利／車両データ依存）
- **リモートエアコン** — 乗車10〜20分前にエアコン起動（18〜32℃を0.5℃単位）。
- **リモートドアロック／ロック忘れ通知**。
- **駐車位置確認** — 広い駐車場で車を探せる。
- **運転診断** — 急発進・急ブレーキ回数などを可視化。
- **車両ステータス一覧** — ガソリン残量などをホームで一目確認。
- **警告灯点灯・定期点検時期のお知らせ** — スマホ／車載ディスプレイへ。

### 1-3. みまもり（共有）
- **みまもるユーザー** — 家族・知人を登録し、リモート機能やアプリサービスを共有。

### スズキコネクトの構造的限界
| 限界 | 内容 |
|------|------|
| **メーカー縛り** | スズキ新車・対応車種のみ。中古・他社車は対象外。 |
| **ハードウェア依存** | 機能のほぼ全てが車載通信機ありきで成立する。 |
| **単機能** | 「車を快適に使う」ことに特化。所有の前後（購入・整備履歴・売却）やコミュニティは守備範囲外。 |
| **クローズド** | スズキ単独のエコシステム。第三者（整備工場・パーツ業者）の参入余地なし。 |

## 2. Trust Car Platform の機能像（実装済みベース）

- **車両管理** — 全メーカー・全年式。車検／保険期限、走行距離、写真、ステータス（使用中／売却／廃車など）。
- **整備記録管理** — 22種のメンテナンスタイプ、費用・工場・部品まで階層記録、統計可視化。
- **OCR自動入力** — 車検証・請求書をML Kitで読み取り、記録へ自動変換。
- **アラート・通知** — 車検／自賠責期限の色分け、燃料・走行距離別のメンテ推奨、FCMプッシュ。
- **SNS・コミュニティ** — 投稿・コメント・いいね・フォロー、ドライブログ（GPS）、アクセサリーシェア。
- **B2Bマーケットプレイス** — 整備工場・パーツショップのリスティング、互換性マッチング、問い合わせスレッド、月次レポート、課金プラン。
- **フリート管理（法人）** — 複数車両一元管理、権限制御（owner/manager/staff/viewer）、CSV出力、一括問い合わせ。
- **AI** — パーツ提案（理由付き複数候補）、メンテスケジュール生成、Claude連携チャット。
- **基盤** — オフライン対応、PDF出力、Crashlytics／Performance／Analytics。

## 3. 直接比較

| 軸 | スズキコネクト | Trust Car Platform |
|----|----------------|--------------------|
| **本質** | テレマティクス（ハード前提） | データ／ソフトウェア（スマホ単体） |
| **対象車** | スズキ新車・対応車種のみ | **全メーカー・全年式・中古含む** |
| **価値の時間軸** | 「乗っている今」（リアルタイム） | **所有ライフサイクル全体**（購入前〜整備〜売却） |
| **緊急通報** | ◎ 自動＋オペレーター | × （ハード非保有） |
| **リモート操作（AC・ロック）** | ◎ | × （ハード非保有） |
| **車両位置・燃料・警告灯** | ◎ リアルタイム | △ 手動記録ベース |
| **整備履歴の蓄積・資産化** | × | **◎ 全車種で蓄積、売却時の価値証明に** |
| **OCR書類デジタル化** | × | ◎ |
| **コミュニティ／SNS** | × | ◎ |
| **B2Bマッチング（整備工場）** | × | ◎ |
| **法人フリート管理** | × | ◎ |
| **AIパーツ提案・チャット** | × | ◎ |
| **課金** | 月額550円（36か月無料） | フリーミアム（個人無料）＋B2Bプラン課金 |

**要点:** ◎が被っている行が一つもない。両者は競合というより**補完関係**にある。
スズキコネクトは「点（今の車両状態）」、Trustは「線（所有の歴史）と面（人・店のつながり）」を扱う。

## 4. 伸ばすべき強み（スズキが構造的に持てない領域）

優先度は既存バックログ（P0）と整合させて設定。

### 強み① 整備履歴の「資産化」 ★最優先
- **狙い:** スズキは新車の車載データしか持てず、**中古車・売却シーンで無力**。Trustは全車種の整備履歴を蓄積でき、これを**売却時の価値証明＝資産**に変えられる。これはスズキコネクトが原理的に到達できない領域。
- **施策:**
  - **愛車タイムラインUI**（既にP0）で所有の歴史を可視化。
  - **「整備記録証明書」PDF**を売却・査定時に提示できる形で出力（PDF基盤は実装済み、内容を査定向けに整える）。
  - 走行距離・整備の一貫性チェックを「改ざんされていない記録」としてアピール。
- **訴求コピー例:** 「あなたの整備履歴が、売るときの説得力になる。」

### 強み② 全車種横断のメンテナンス予測 ★最優先
- **狙い:** スズキは自社車種の警告灯のみ。Trustは**全メーカー × 走行距離 × 過去履歴**から先回り提案できる。
- **施策:**
  - **ホーム「AIからの提案」セクション**（既にP0）— 「○○km走りました。次の候補は△△です（理由）」。
  - 燃料タイプ別スケジュール（実装済みのMaintenanceScheduleService）をホーム導線に接続。
- **設計思想を守る:** AIは判断せず、理由付きで選択肢を整理するだけ。

### 強み③ コミュニティ × マーケットの一気通貫導線
- **狙い:** スズキにはコミュニティもマーケットもない。Trustは「同じ車種オーナーのつながり → パーツ提案 → 整備工場マッチング」を**一本の動線**にできる。
- **施策:**
  - SNSフィードのアクセサリー投稿から、互換パーツ／施工可能な工場へワンタップ遷移。
  - ドライブログ → スポット → 近隣の整備・洗車工場レコメンド。
- **これがネットワーク効果の源泉**（ユーザーが増えるほど価値が上がる）になり、模倣困難な堀になる。

### 強み④ 家族・グループ共有（スズキ「みまもり」への対抗）
- **狙い:** スズキの「みまもるユーザー」に相当する体験を、**ソフトウェアだけで・全車種で**実現。
- **施策:**
  - 車検／整備予定・期限アラートを家族で共有（既存の通知・権限基盤を流用、フリートの権限モデルが転用可能）。
  - 「ロック忘れ」のようなハード機能は追わず、**期限・予定・履歴の共有**という非ハード領域に絞る。

### 強み⑤ 法人フリート（完全に無風の市場）
- **狙い:** スズキコネクトは法人フリート管理を提供しない。Trustは権限制御・CSV・一括問い合わせを実装済み。**B2B収益の主軸**として磨く。

---

## 5. やらないこと（戦略的撤退ライン）

- **テレマティクス機能の自前実装を追わない** — リモートAC／自動緊急通報は車載ハードが必須で、ソフト単体では勝てない。OBD-II連携（将来ロードマップ）も**コア戦略には据えない**（ソフトの堀を薄める）。
- スズキの「点」の強みを真似るのではなく、Trustの「線・面」の強みに資源を集中する。

---

## 6. ポジショニング一言まとめ

> **スズキコネクトは「乗っている間」を快適にする。Trust Car Platform は「所有のすべて」に寄り添う。**
> 全メーカー対応・整備履歴の資産化・人と店のつながり —— これらはハードを持つOEMには真似できない、ソフトウェア・プラットフォームだけの強み。
