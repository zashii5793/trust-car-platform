## 1. プロダクト 1 行紹介

**ZAXEL Learning** は、現場で鍛えたビジネススキル（コミュニケーション・思考法・組織論）を、スマホで遊べる一問一答クイズで身につける学習サービスです。

## 2. ターゲット（B2C 個人）

| セグメント | 利用シーン | ペイン |
| --- | --- | --- |
| 20代後半〜30代の若手ビジネスパーソン | 通勤・昼休み・寝る前 | 「研修を受けても定着しない」「本を読む時間がない」「実務で詰まる」 |
| プレイングマネージャー（30〜40代） | 部下指導の前にネタ確認 | 「教える立場として体系化したい」「自分の感覚を言語化したい」 |
| コンサル・士業の若手 | 自己研鑽 | 「先輩の暗黙知を効率よく吸収したい」 |

**初期フォーカス**: 「過去にセミナーを受講してくれた延長線上の社会人」＋「SNSで興味を持った類似ペルソナ」。

## 3. プライシング（v0.1）

| プラン | 価格 | 想定提供価値 |
| --- | --- | --- |
| Free | ¥0 | 1日3問まで無料 / FREEデッキは全問プレイ可能 / ストリーク・スコア記録 |
| Pro Monthly | ¥480 / 月 | すべてのPROデッキ開放 / 無制限プレイ / 復習モード / クラウド同期 |
| Pro Annual | ¥3,980 / 年 | 月額の31%OFF（実質¥332/月） |
| Team（将来） | ¥300 / 人・月（10名〜） | チーム進捗ダッシュボード / 管理者向け分析 |

### 価格設計の根拠
- スタディサプリ ENGLISH（月額3,278円）、Duolingo Plus（月額1,300円）と比較して**圧倒的に始めやすい価格帯**に設定
- 「コーヒー1杯分」（=¥480）が記憶想起しやすい心理的アンカー
- 年額は2か月分お得を演出（解約防止効果）

### 価格テスト案
- T0: ¥480 / 月（初期ローンチ価格、~3か月）
- T1: 体験者の継続率と解約率を見て ¥580 / ¥680 への引き上げ検討
- T2: 法人プラン投入時に個人プランも見直し

## 4. 無料ライン設計（フリーミアム）

### 制限設計の3軸
1. **デッキ単位の制限**: FREE / PRO バッジで明示
2. **プレイ回数制限**: 1日3プレイまで（FREE / PROデッキ問わず体験は可能）
3. **機能制限**: 復習モード・クラウド同期は Pro 限定

### 「無料でどこまで触れるか」のチューニング指針
- 入口は限りなく緩く（Pro 必須にしない）
- 「3問やると気持ちよく終わる」体験を Free でも提供
- 4問目以降に Paywall を見せて転換率を計測

### 想定 Free→Pro 転換率
- 初期: 1〜3%（ベンチマーク：Duolingo 2-3%, スタディサプリ 5-8%）
- 目標: 6か月以内に 5%、1年以内に 8%

## 5. 収益モデル（粗試算）

### 仮定
- 月間アクティブユーザー (MAU): 1,000 → 5,000 → 20,000（3か月毎）
- 有料転換率: 3%
- ARPU（有料）: ¥480
- 解約率: 月8%（フリーミアムSaaS平均）

### MRR（月次経常収益）試算
| 時点 | MAU | 有料会員 | MRR |
| --- | ---: | ---: | ---: |
| ローンチ +1か月 | 1,000 | 30 | ¥14,400 |
| +3か月 | 5,000 | 150 | ¥72,000 |
| +6か月 | 10,000 | 300 | ¥144,000 |
| +12か月 | 20,000 | 600 | ¥288,000 |

### 補助的な収益源候補（v1.0 以降）
- **法人向けプラン**: チームダッシュボード付き ¥300/人・月
- **企業研修パッケージ**: 既存セミナー資料のクイズ化代行 / カスタムデッキ作成 (¥30万〜)
- **アフィリエイト**: 元ネタになった書籍へのリンク（Amazon等）
- **広告**: あえて入れない（UX重視）

## 6. コンテンツ供給戦略

### コンテンツの源泉
1. **既存セミナー資料**: Google Drive にある約50本のPDF（コミュニケーション、生産性、組織論、戦略、AI活用 etc.）
2. **新規執筆**: 月2デッキずつ追加（10〜15問/デッキ）
3. **ユーザー投稿**（v2.0 構想）: コミュニティで問題を投稿、運営審査で公開

### コンテンツ生成パイプライン（運営者向け）
1. PDF をアップロード or テキスト貼り付け
2. AI（Claude等）で初稿クイズ生成（10問）
3. 著者（=セミナー講師）が校正・解説強化
4. JSON / DB 登録 → アプリに反映

→ このパイプライン自体を自動化することで、月10デッキでも余裕で運用可能。

### 初期 4 デッキ（実装済み）
- 💬 コミュニケーション理論（FREE）
- 📚 本を読んで書こう（FREE）
- 🧠 抽象化力を鍛える（PRO）
- 🎯 エンゲージメント振り返り（PRO）

### ロードマップ（次の8デッキ案）
- 仕事の依頼の仕方（FREE）
- 進捗報告のお作法（FREE）
- 会議進行のお作法（PRO）
- 生産性向上の時間術（PRO）
- 価値提供の本質（PRO）
- 新規事業の企画力（PRO）
- 戦略会計の入口（PRO）
- AI 時代の働き方（PRO）

## 7. グロース（獲得）戦略

### 0→1 フェーズ（最初の100人）
- セミナー受講者・知人ネットワークへの招待
- Twitter / Threads でクイズを1問ずつポストし、続きはアプリへ誘導
- note 記事で「研修で教えてきたエッセンス」を発信し、末尾でアプリ案内

### 1→10 フェーズ（1000人）
- 「結果シェア機能」を実装（"私のスコアは 90% 🎉" を画像でX投稿）
- 法人案件で導入したクライアントの担当者を個人ユーザーとしても定着させる
- インフルエンサー（マネジメント系・自己啓発系）へのアプローチ

### 10→100 フェーズ（10000人）
- SEO（「会議 進行 コツ」「依頼 仕方」等のロングテール）
- App Store / Google Play 公開（PWA → ネイティブ化）
- 法人パッケージで企業内浸透 → 個人での継続利用へ

## 8. 競合・差別化

| 競合 | 強み | ZAXELの差別化 |
| --- | --- | --- |
| Duolingo | ゲーミフィケーション完成度 | 「ビジネススキル特化」「日本人ペルソナ・日本語UX」「現場のリアルなネタ」 |
| スタディサプリ ENGLISH | 法人導入実績 | 英語ではなく**仕事スキル**、月額1/6の価格 |
| グロービス学び放題 | コンテンツ網羅性 | 動画ではなく**クイズ**で軽快、月額1/10、隙間時間に最適化 |
| Studyplus / Anki | 自学習ツール | コンテンツ供給込み、初心者でもすぐ始められる |

**ユニーク・バリュー・プロップ**：
> 「読まなくていい。動画も観なくていい。1問15秒で、現場の知恵が体に染み込む。」

## 9. リスクと対策

| リスク | 影響 | 対策 |
| --- | --- | --- |
| コンテンツ枯渇 | 継続率低下 | AI支援パイプラインで生成効率化、ユーザー投稿で長期スケール |
| 競合参入（特に大手） | 価格競争 | 「現場の生々しさ」「人間味のある解説」で差別化 |
| 有料転換が伸びない | 売上不足 | 法人向けパッケージで補完、年額プランで LTV 改善 |
| プラットフォーム（App Store手数料30%） | 利益率悪化 | Web (PWA) 優先、ネイティブは後発 |
| 著作権（書籍引用） | 法的リスク | 一次資料は講師オリジナル、書籍は要約・参考レベルに留める |
