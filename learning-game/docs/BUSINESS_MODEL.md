## 1. プロダクト 1 行紹介

**ZAXEL-Learning** は、現場で鍛えたビジネススキル（コミュニケーション・思考法・組織論）を、スマホで遊べる一問一答クイズで身につける学習サービスです。

## 3. プライシング（v0.2 — 買い切りモデル）

| 商品 | 価格 | 想定提供価値 |
| --- | --- | --- |
| FREEデッキ | ¥0 | 全問・無制限プレイ無料 / ストリーク・スコア記録 |
| PROデッキ（1個） | ¥240 / 買い切り（永続） | 該当デッキを永続アンロック。全問プレイ＋解説 |
| PROデッキ全部入りバンドル | ¥600 / 買い切り（永続） | 現時点で公開中のPROデッキを一括アンロック |
| 法人ライセンス（将来） | 個別見積（10名〜） | 全社員に全PROデッキ + チーム進捗ダッシュボード |

### なぜサブスクではなく買い切りか

1. **解約忘れストレスがない** — ユーザーの「契約してる感」を消し、必要な分だけ気軽に買えるUX
2. **コンテンツ＝資産という性質との整合** — 「本を1冊買う」体験に近い納得感
3. **ユーザー獲得後の継続営業コストが低い** — チャーン管理よりも追加デッキの魅力訴求に集中できる
4. **競合サブスク（Duolingo, スタディサプリ）と差別化** — "サブスク疲れ" 層を取り込める

### 価格設計の根拠

- 1デッキ ¥240 = コンビニコーヒー1杯〜ランチ1食より安い心理的アンカー
- バンドル ¥600 = 「PROデッキ全部入り」が文庫本1冊以下
- 書籍要約サービス（flier 月額2,200円）と比較しても圧倒的に始めやすい
- 「気に入ったテーマだけ買い、合わなければ放置」を許容する価格

### 価格テスト案

- **T0（ローンチ〜3か月）**: 1デッキ ¥240 / バンドル ¥600
- **T1（3〜6か月）**: 購入率・LTV を見て、¥290 / ¥780 への引き上げをABテスト
- **T2（6か月〜）**: 高単価デッキ（ボリューム多 or 専門特化）を ¥480 で投入

## 4. 無料ライン設計（フリーミアム）

### 制限設計の2軸

1. **デッキ単位の制限**: FREE / PRO バッジで明示
2. **PROデッキのお試し**: 最初の **3問** はお試しでプレイ可、4問目以降は購入

### 「無料でどこまで触れるか」のチューニング指針

- 入口は限りなく緩く — FREEデッキは無制限プレイで完全開放
- PROデッキの3問お試しで「解説の質」「自分との相性」を体験させる
- 4問目で Paywall を見せて転換率を計測（A/B: 2問 / 3問 / 5問）

### 想定 Free→Paid 転換率（買い切りモデル）

- 初期: アクティブユーザーの **3〜5%** が少なくとも1デッキ購入
- 目標: 6か月以内に 8%、1年以内に 12%
- バンドル購入比率: 個別購入者のうち 20〜30% がバンドルにアップグレード

## 5. 収益モデル（粗試算）

### 仮定

- 月間アクティブユーザー (MAU): 1,000 → 5,000 → 10,000 → 20,000
- 1か月あたりの新規購入率: 4%
- 1回あたりの平均購入金額（ARPU per buyer）: ¥360（個別 ¥240 と バンドル ¥600 の加重平均）
- 既存購入者の追加デッキ購入率: 月1.5%（リピート購入）

### 月次売上試算

| 時点 | MAU | 新規購入者 / 月 | リピート売上 | 月次売上 |
| --- | ---: | ---: | ---: | ---: |
| ローンチ +1か月 | 1,000 | 40 | — | ¥14,400 |
| +3か月 | 5,000 | 200 | ¥1,800 | ¥73,800 |
| +6か月 | 10,000 | 400 | ¥10,800 | ¥154,800 |
| +12か月 | 20,000 | 800 | ¥36,000 | ¥324,000 |

> サブスクモデル（v0.1）と比較すると初動の MRR は同等だが、買い切りモデルは **チャーン0** のため累積LTVが時間とともに頭打ちにならない強みがある（=新規デッキ投入のたびに既存顧客からも収益が立つ）。

### 補助的な収益源候補（v1.0 以降）

- **法人向け一括ライセンス**: 全社員 × 全PROデッキで個別見積（¥30万〜）
- **企業研修パッケージ**: 既存セミナー資料のクイズ化代行 / カスタムデッキ作成 (¥30万〜)
- **講師向けデッキ販売プラットフォーム**: 第三者講師がデッキを出品し、レベニューシェア (運営30%取り)
- **アフィリエイト**: 元ネタになった書籍へのリンク（Amazon等）
- **広告**: あえて入れない（UX重視）

## 6. コンテンツ供給戦略

### コンテンツの源泉

1. **既存セミナー資料**: Google Drive にある約50本のPDF（コミュニケーション、生産性、組織論、戦略、AI活用 etc.）
2. **新規執筆**: 月2デッキずつ追加（10〜15問/デッキ）
3. **講師ネットワーク**（v2.0 構想）: 外部講師のデッキ出品プラットフォーム化

### コンテンツ生成パイプライン（運営者向け）

1. PDF をアップロード or テキスト貼り付け
2. AI（Claude等）で初稿クイズ生成（10問）
3. 著者（=セミナー講師）が校正・解説強化
4. JSON / DB 登録 → アプリに反映 → 価格設定（標準 ¥240）

→ このパイプライン自体を自動化することで、月10デッキでも余裕で運用可能。

### 初期 4 デッキ（実装済み）

- 💬 コミュニケーション理論（FREE）
- 📚 本を読んで書こう（FREE）
- 🧠 抽象化力を鍛える（PRO ¥240）
- 🎯 エンゲージメント振り返り（PRO ¥240）

### ロードマップ（次の8デッキ案）

- 仕事の依頼の仕方（FREE）

## 7. グロース（獲得）戦略

### 0→1 フェーズ（最初の100人）

- セミナー受講者・知人ネットワークへの招待
- Twitter / Threads でクイズを1問ずつポストし、続きはアプリへ誘導
- note 記事で「研修で教えてきたエッセンス」を発信し、末尾でアプリ案内

### 1→10 フェーズ（1000人）

- 「結果シェア機能」を実装（"私のスコアは 90% 🎉" を画像でX投稿）
- 法人案件で導入したクライアントの担当者を個人ユーザーとしても定着させる
- インフルエンサー（マネジメント系・自己啓発系）へのアプローチ
- バンドル購入の動機付け（「今だけ ¥800 → ¥600」期間限定値引きキャンペーン）

### 10→100 フェーズ（10000人）

- SEO（「会議 進行 コツ」「依頼 仕方」等のロングテール）
- App Store / Google Play 公開（PWA → ネイティブ化、ただしApp内課金30%手数料で価格見直し）
- 法人パッケージで企業内浸透 → 個人での継続利用へ
- ギフト購入機能（「このデッキを同僚に贈る」¥240 のソーシャル拡散）

## 9. リスクと対策

| リスク | 影響 | 対策 |
| --- | --- | --- |
| コンテンツ枯渇 | リピート購入低下 | AI支援パイプラインで生成効率化、月2デッキ以上のペース維持 |
| 1回購入で離脱（リピート無し） | LTV低迷 | プッシュ通知での新デッキ告知、シリーズ化（「コミュ理論 II」） |
| 競合参入（特に大手） | 価格競争 | 「現場の生々しさ」「人間味のある解説」で差別化 |
| 買い切りモデルの収益不安定さ | キャッシュフロー | 法人向けライセンス収益で底上げ、年末セールなど季節販促 |
| プラットフォーム（App Store手数料30%） | 利益率悪化 | Web (PWA) 優先、ネイティブは後発で価格を ¥340 など調整 |
| 著作権（書籍引用） | 法的リスク | 一次資料は講師オリジナル、書籍は要約・参考レベルに留める |
